# Online Store Product Model

This project is a simple Java implementation of a **Product** class for an online store, accompanied by a UML class diagram created in **Visual Paradigm**.

The purpose of the assignment is to:
1. Create classes that describe objects (a Product in an online shop).
2. Draw class diagrams using a modeling tool.
3. Implement the classes in Java.
